# Lost & Found MVP

Simple academic project to post and view lost/found items.

## Prerequisites
- Node.js
- MongoDB (running locally on port 27017)

## Setup & Run

### 1. Database
Ensure MongoDB is running.

### 2. Backend
Open a terminal in the root and run:
```bash
cd server
npm install
npm run dev
```
Server runs on `http://localhost:5000`.

### 3. Frontend
Open another terminal and run:
```bash
cd client
npm install
npm run dev
```
Frontend runs on `http://localhost:5173` (or similar).

## Features
- View Lost Items
- View Found Items
- Post Lost Item
- Post Found Item
